CDH研究平台简介CDH (Coronary heart diseases)

冠状动脉粥样硬化性心脏病是冠状动脉血管发生动脉粥样硬化病变而引起血管腔狭窄或阻塞，造成心肌缺血、缺氧或坏死而导致的心脏病，常常被称为“冠心病”。

高效CHD研究工具，提供病因前瞻性研究的数据追踪系统。


针对前瞻性队列研究的劣势，本项目试图通过建立人群病学基线资料的非关系型数据库，详尽记录每位患者（实验对象）的历史基线资料表与历史CCTA检查表。
通过对数据库的严谨设计，降低CHD前瞻性队列研究数据存储空间的耗费。
在CHD病因研究中，前瞻性队列研究从因到果，与对照研究等传统方式相比在医学研究中可信度高，偏倚少。但因其研究的结局需要一段时间的随访观察才能得到，因此耗时耗力，成本较高。

数据可视化处理


对数据进行可视化处理，直观对数据进行对比，降低研究时间成本。 本项目最终希望通过数据库的设计与平台的建立，为冠心病前瞻性队列研究降低成本，推动前瞻性队列研究的发展
